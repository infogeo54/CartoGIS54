These sections will quickly introduce you the interface and show you how to use it to manage your geodata.

Header

It's the top part of the interface where you could find buttons that, like the one you have clicked to display this help, will toggle modals added by the developers.

Menu

On the left of the window, the Menu is composed of the Layer list and the Legend.

Layer list

This is the part of the menu that lists project's layers.

Clicking a layer name from the Layer List will open it's Legend.

Legend

The Legend is used on one hand to display a layer's icons, and on the other hand to start creating a new feature.

Indeed, clicking an icon will allow you to click on the Map in order to create a new Feature.

Map

Completely interactive, the Map is the main part of the application.

It is used to display, create or edit data. You can interact with existing features by clicking on their representation.

Form

The Form is the last element of the interface. It's a simple way to provide or edit features' data.
